#BidBig

Steps to run the application
Requirements:
1) OS: Ubuntu 18 or Mac OSX High Sierra, 8GB Ram
2) Docker and Docker Compose latest version
3) Java 1.8
4) Maven
5) React 16.6.3
6) Nodejs, NPM

For Backend
Navigate to root folder
First Start
1.    Run “. en.sh”
2.    Run “./start.sh”
Restart
1.    Run “. en.sh”
2.    Run “./restart.sh”

For Frontend
Navigate to the root folder
 To run locally

Run “npm install”
Run “npm run start”


Navigate to https://localhost:1500
